What is DOM & what is the purpose of DOM?

DOM stands for Document Object Model. As per name, we can easily understand that - "It turn documents data into Object & we can access them by Node list or other ways." It works with one of the fundamental programming language called JavaScript. It is a scripted language. DOM basically provides a structured representation of the web document or HTML. It turn the whole HTML file into several Nodes. Every Tag or elements inside of the HTML file get converted into Nodes. As a result, we can manipulate the HTML file by capturing the nodes of the documents.By manipulating the DOM, we can change the content and appearance of a web page dynamically, in response to user actions or events. Like : MouseClick, Mouse Hover,Keypress etc.

How will you select HTML elements using DOM?
Name the DOM methods.

We can select elements from DOM by FIVE ways. Some of those will give us a individual element and some will give us a group of elements . The Five ways we can select the elements are given below :
1. getElementById: This will give us a specific element.
Example : const myElement = document.getElementById("element-ID");
2. getElementsByClassName: This will select all the elements containing same class name & will give us an array like object of those elements. Which can contain bunch of elements as a group. example : HTMLCollection(...) [................].
Example : const myElements = document.getElementsByClassName("class-name");
3. getElementsByTagName:Similar as the previous. This will select all the elements containing same Tag name & will give us an array like object of those elements. example : HTMLCollection(...) [................].
Example : const myParagraphs = document.getElementsByTagName("h1");
4. querySelector:This will give us the 1st element of a NodeList. This will have the whole html tag which contain the specific class,id,name,index. We have to use this like a CSS Property.
Example : const myElementID = document.querySelector("#name-ID");
5. querySelectorAll:This will give us a NodeList. like an array like object. It will contain all the elements containing that specific class,id,name or index. NodeList(...) [................].
Example : const myElementsByClass = document.querySelectorAll(".name-class");

What is Event Bubble..??

Event bubbling is a behavior in the Document Object Model which stands for (DOM). Basically event bubbling is a case when we call a function or event it executes all the function or event from the target or child elements to parent elements. Sequence can defer. In this particular behavior there are 3 stages.
1. Capture.
2. Target.
3. Bubble.
At first it travels to the area of function or event to execute the event handler. The travel movement can defer but it basically Travels from parent child to child element. Secondly, it capture the target and then execute it.
Third, after executing the child element it travel back to the event function and in the traveling back it executes all the event handler it gets on its way.
To Stop bubbling we can use STOP PROPAGATION or STOP IMMEDIATE PROPAGATION.

What is a callback function
& why will you use it?

Mainly a Callback is a function passed as an argument to another function. This allows a function to call another function in itself. A callback function can run after another function has finished.
A callback's primary purpose is to execute codes when user performs any event, then it response. These events which can be performed by user, such as mouse clicks or typing. Mostly With a callback, we can set a instruction for the application program to execute that code every time the user clicks a key on the keyboard button.
In JavaScript, parameters are used to pass information to a function. A callback function typically takes one or more parameters that are passed to it by the calling function. These parameters provide data or context the callback function can use to perform some operation or generate a result.
